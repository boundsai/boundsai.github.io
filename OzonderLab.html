Detection of Vulnerabilities in Post-Quantum Cryptography Algorithms with Deep Learning (Completed)

Project Name: Detection of Vulnerabilities in Post-Quantum Cryptography Algorithms with Deep Learning

Turkish Title: “Derin Öğrenme ile Kuantum Sonrası Kriptografi Algoritmalarının Kırılganlıklarının Tespiti”

Managed By: Şener Özönder

Supported By: TÜBİTAK BİLGEM

Start Date: 03/06/2024

End Date: 30/05/2025

Status: Completed

Project Summary: This project tests transformer-based approaches for extracting secret keys in order to identify vulnerabilities of lattice-based post-quantum cryptography algorithms against deep learning. By evaluating a wide range of popular transformer models, the project aims to identify safe regions in the parameter space of encryption algorithms that cannot be broken by quantum computers.
